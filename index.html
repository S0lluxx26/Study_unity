COMMON BLOCKERS

          
Fix the first five problems.

          
My script does not appear in Add Component
Fix every red Console error first. Confirm the file name matches the public class name and that the class inherits MonoBehaviour.

          
My player falls through the floor
Give the player a Rigidbody and Collider. Give the floor a Collider. Confirm neither Collider is accidentally set as a trigger.

          
Input does nothing
Confirm the Input System package is installed, the action map is enabled, PlayerInput references the action asset, and the callback name matches the action.

          
Everything moves at a different speed
Scale frame-based movement by Time.deltaTime. Apply Rigidbody forces in FixedUpdate rather than Update.

          
The Web build opens to a blank page
Test with Build and Run or a local web server—not by double-clicking the HTML file. Then inspect the browser console and server compression headers.
